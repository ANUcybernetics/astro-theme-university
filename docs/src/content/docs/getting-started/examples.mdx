The quickest way to get started is to copy the starter example in the `examples/` directory. It's a self-contained Astro project with the theme pre-configured, a CI config for GitLab Pages-style deployment, and example content you can replace with your own.

## Quick start

Clone the repository and copy the example into a new directory:

```bash
git clone https://github.com/ANUcybernetics/astro-theme-university.git
cp -r astro-theme-university/examples/base my-site
cd my-site
pnpm install
pnpm dev
```

## What's in the base example

A minimal themed site with a single `pages` content collection. Good for project sites, research group pages, and general content.

Includes:

- `PageLayout` for MDX pages with optional hero images
- a `pages` content collection for markdown/MDX pages
- `site-config.ts` for site name and navigation

## Deployment

The example ships with a `.gitlab-ci.yml` for GitLab Pages-style deployment: commit `pnpm-lock.yaml`, push to `main`, and CI builds and deploys automatically. The config reads `SITE_URL` and `BASE_PATH` from CI variables --- set them to match your host, or adapt the config (see [deploying](/docs/guides/deploying/)) for other platforms.

## Next steps

Once you have the example running, edit `src/site-config.ts` to set your site name and navigation links. See [configuration](/docs/getting-started/configuration/) for details on all the available options.
