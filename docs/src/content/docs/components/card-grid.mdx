The CardGrid component arranges Card components in a responsive grid. It supports 2 or 3 column layouts and collapses gracefully on smaller screens.

```astro
---
import CardGrid from "astro-theme-university/components/CardGrid.astro";
import Card from "astro-theme-university/components/Card.astro";
---
```

## Props

<PropsTable component="components/CardGrid" />

The component accepts a default slot, intended for Card components.

## Notes

While CardGrid is designed to work with Card components, it will accept any content in its slot. The grid uses CSS to handle responsive behaviour --- cards will stack to a single column on narrow viewports regardless of the `columns` setting.
