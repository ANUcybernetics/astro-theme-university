The FillInTheBlank component, built with [Svelte 5](https://svelte.dev/), renders a code template with interactive input fields. Students fill in the blanks and get instant feedback --- correct answers turn green and lock in place.

```astro
---
import FillInTheBlank from "astro-theme-university/components/FillInTheBlank.svelte";
---

<FillInTheBlank client:load template={'let greeting = "<name:hello>";'} />
```

## Template syntax

Placeholders use `<name:answer>` format within the template string:

- `<name:answer>` --- `name` is shown as placeholder text, `answer` is the expected value
- Multiple blanks per template are supported
- The input field auto-sizes to the answer length

## Notes

Unlike P5Sketch, this component works with `client:load` (no browser-only APIs needed). Correct answers are marked read-only to prevent accidental changes. The completion counter updates live as students work through the exercise.
