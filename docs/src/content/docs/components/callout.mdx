The Callout component draws attention to important information using a styled [`<aside>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/aside) element with [`role="note"`](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/note_role). Four visual variants are available --- tip, info, warning, and error --- each with distinct styling to convey the appropriate level of urgency.

## Markdown syntax

In MDX and markdown files, use the `:::` container directive syntax instead of importing the component directly. See [markdown extensions](/docs/guides/markdown-extensions#callouts) for the full syntax and live examples.

## Component syntax

You can also use the Callout component directly:

```astro
---
import Callout from "astro-theme-university/components/Callout.astro";
---
```

## Notes

The callout renders as an `<aside>` with `role="note"`, which screen readers will treat as supplementary content. Keep callout text concise --- if you find yourself writing multiple paragraphs, the content probably belongs in the main body instead.
